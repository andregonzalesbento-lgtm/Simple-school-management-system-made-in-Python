# Python School System

Sistema escolar simples desenvolvido em Python.

## Funcionalidades

- Cadastro de alunos
- Cálculo automático de média
- Controle de faltas
- Reprovação por nota ou por falta
- Ranking da turma
- Pesquisa de aluno
- Salvamento em arquivo JSON

## Tecnologias utilizadas

- Python
- JSON
- Terminal (CLI)

## Objetivo

Projeto desenvolvido para praticar lógica de programação e estrutura de dados em Python.
